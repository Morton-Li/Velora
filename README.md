# Velora

Velora 是一款为 macOS 设计的下载工具。

它基于 `aria2` 实现下载能力，并以原生 macOS 应用的方式呈现：简单、明确添加链接，选择保存位置，查看下载进度，然后让任务安静地继续运行。

## 设计理念

Velora 希望把下载工具做得简单、直接、可靠。不追求复杂功能堆叠，优先保证常用操作清晰准确，任务状态容易理解，行为符合用户预期。

## 技术栈

- SwiftUI
- macOS App Sandbox

## 系统要求

- macOS 14.0 or later
